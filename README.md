# ✂️ TwoCut - Isometric Barbershop Management Game

![Unity Version](https://img.shields.io/badge/Unity-6000.3.10f1-blue.svg)

> *"Mahallenin yeni kuaförünü açtınız — ama açtığınız tek şey dükkan değil, ortağınızla omuzladığınız kocaman bir borç. Makasları kapın, koltukları hazırlayın ve zamana karşı yarışın."*

**TwoCut**, izometrik açıda geçen, dükkan yönetimi, zamana karşı vardiya ve co-op koşuşturma oyunudur.

---

## ✨ Öne Çıkan Özellikler

- ⏳ **Zamana Karşı Vardiyalar & Borç Taksit Sistemi**: Her gün ödenmesi gereken günlük borç taksitleri. Taksit ödenmezse borç birikir ve iflas riski doğar!
- 💈 **4 Temel Hizmet**: Saç kesimi, yıkama, boyama ve masaj hizmetleri.
- 🧹 **Dağınıklık & Temizlik Mekaniği**: Saç kesimi sonrası yerlere dökülen saç kırıklarını süpürme. Kirli ortam müşteri sabrını düşürür!
- 🛒 **Dükkan Büyütme & Ekipman Alma**: Kasa kârıyla Altın Makas, Masaj Koltukları, Otomatik Temizlik Robotları ve Yardımcı Çıraklar alma.
- 🌐 **Online Co-op Altyapısı**: Çok oyunculu (Multiplayer) mimariye uygun yerel ve ağ oyuncusu yapılandırması.

---

## 🎮 Kontroller

| Tuş | İşlev |
| :--- | :--- |
| **`W`, `A`, `S`, `D`** | Karakter Hareketi |
| **`Sol Shift`** | Depar / Hızlı Koşma |
| **`E`** | Eşya Al / Bırak |
| **`F`** | İşlem Yap (Kesim/Yıkama/Masaj/Süpürme) |
| **`G`** | Yere Eşya Bırakma |

---

## 🛠️ Kurulum & Geliştirme (Developers)

1. Projeyi bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/USERNAME/TwoCut.git
   ```
2. **Unity Hub**'ı açın ve projeyi Unity 6 (`6000.3.10f1` veya üstü) ile projeleriniz arasına ekleyin.
3. Projeyi açın, üst menüden `TwoCut -> Clean & Build Single Player Scene` seçeneğine basarak sahneyi otomatik kurun ve **Play (▶)** butonuna basın!
